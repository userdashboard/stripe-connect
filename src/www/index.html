Build web apps faster with Dashboard

A web app for your web app

Dashboard provides a complete UI and API for creating, maintaining, and administrating your user accounts and sessions.

Using Dashboard lets you write a smaller, more focused web app with a full-featured accounting system:

  • API for users and administrators
  • UI for users and administrators
  • credentials encrypted with bcrypt
  • multiple storage options
  • does not require JavaScript
  • mobile and desktop friendly

Stripe Connect module

This module adds a UI and API endpoints for administrators and users to create Stripe Connect accounts using the Stripe.com APIs..

The Connect module may be used in conjunction with the Subscriptions module to allow subscribing to the Connect accounts' products.
